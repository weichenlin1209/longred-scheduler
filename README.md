## 使用教學
打開表單回覆的試算表，點擊 `檔案` >> `下載` >> `.csv`。下載之後把檔案拉到跟這個程式同一個資料夾裡，並把檔名改成 `data.csv`。接下來根據不同作業系統會有不同的作法，分別用 `Linux`、`MacOS`、`Windows`講解。

### Linux
打開`bash`，cd 到你目前資料夾。輸入：  

```
./execute.sh schedule.c  
```

就會看到 `result.csv` 出現在資料夾裡面
